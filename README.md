🎮 Tic Tac Toe Game

Welcome to my Tic Tac Toe project!

This is a simple console-based Tic Tac Toe game that I developed using the C programming language. I created this project to strengthen my understanding of programming fundamentals and to practice applying concepts such as arrays, functions, loops, conditional statements, and user input handling in a real-world application.

The game allows two players to enter their names, choose their playing symbols, and compete in a classic Tic Tac Toe match. It automatically checks every move, determines the winner, and also detects draw situations.

🎮 How the Game Works

1. The game starts with a welcome message.
2. Both players enter their names.
3. Player 1 chooses either X or O.
4. Player 2 automatically receives the remaining symbol.
5. Players take turns selecting positions from 1 to 9.
6. After every move, the game updates the board.
7. The program continuously checks for a winning combination.
8. If a player completes a row, column, or diagonal, the game announces the winner.
9. If all positions are filled without a winner, the game ends in a draw.
